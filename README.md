# LIGO Open Data Workshop 2025

This repository contains notebooks completed during the LIGO Open Data Workshop 2025.

The material covers several core techniques used in gravitational-wave astronomy, including:

* Accessing and visualizing public LIGO/Virgo data
* Time-frequency analysis with Q-transforms
* Waveform generation
* Matched filtering
* Signal consistency tests
* Bayesian parameter estimation
* Posterior analysis

## Contents

### Module 1: Data Access and Visualization

* 1.1 Discovering Open Data
* 1.2 Open Data Access with GWPy
* 1.3 Q-transforms with GWPy
* 1.4 Generating Waveforms

### Module 2: Signal Detection

* 2.1 Matched Filtering Introduction
* 2.2 Matched Filtering in Action
* 2.3 Signal Consistency and Significance

### Module 3: Parameter Estimation

* 3.1 Introduction to Parameter Estimation
* 3.2 Parameter Estimation for Compact Objects
* 3.3 Discovering and Using Published Posteriors

## Disclaimer

Most notebooks originate from the LIGO Open Data Workshop. Additional comments, notes, and modifications were added as part of my learning process.
